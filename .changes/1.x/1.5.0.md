
## 1.5.0 - 2026-01-31

> [!NOTE]
> Enhanced code with help of [Mago](https://github.com/carthage-software/mago)

### Added

- new `Bartlett\Sarif\Contract\HelpUriCaseConverterInterface` to handle `helpUri` generation for each converter
- new `Bartlett\Sarif\Contract\NormalizerInterface::isSupportedFormat` method
to check if the format is supported by the current normalizer
- support for Symfony 8 components

### Changed

- extract `helpUri` generation from ecs source code converter
to specialized `Bartlett\Sarif\Converter\Case\EcsHelpUriCaseConverter` class
- rewrites custom autoloader to use only dependencies from each converter (no additional required)
- revisit how to get tool version with [Composer runtime API](https://getcomposer.org/doc/07-runtime.md)
- `input-format` option of `convert` command is now case-insensitive

**Full Changelog**: [1.4.0...1.5.0](https://github.com/llaville/sarif-php-converters/compare/1.4.0...1.5.0)
